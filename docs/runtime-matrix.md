# Runtime Matrix

| Runtime field | Meaning |
|---|---|
| `runtime_name: apple-core-ai` | Model artifact targets Apple Core AI |
| `format: aimodel` | Artifact is an Apple Core AI `.aimodel` |
| `source_group: zoo` | Community-port model from upstream zoo |
| `source_group: official` | Official-recipe conversion from upstream official directory |
| `stock_runtime: true` | Runs on stock runtime according to upstream classification |
| `custom_kernel: true` | Uses or depends on a custom Metal kernel path |
| `patch_required: true` | Requires a patch or non-standard workaround |
