# Core AI Catalog — Examples

Real-world Swift snippets and integration patterns for Apple Core AI models.

## Available examples

| Example | Model | Task | Device |
|---|---|---|---|
| [ocr-swiftui](./ocr-swiftui/README.md) | Unlimited-OCR | On-device OCR | iPhone/iPad/Mac |
| [vlm-chat](./vlm-chat/README.md) | Qwen3-VL 2B | Vision-language chat | iPhone/iPad/Mac |
| [embeddings-rag](./embeddings-rag/README.md) | EmbeddingGemma 300M | On-device embeddings + RAG | iPhone/iPad/Mac |

## Prerequisites

```bash
# Install the catalog CLI
pip install coreai-catalog

# Install a model (downloads .aimodel bundle from Hugging Face)
coreai-catalog install unlimited-ocr
```

## Using snippets

Each example includes a copy-pasteable Swift integration that loads the
`.aimodel` bundle via CoreAIKit. The snippet generated by `coreai-catalog install`
goes to `~/.coreai-catalog/models/<model-id>/snippet.swift`.
