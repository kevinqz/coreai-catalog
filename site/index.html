About Core AI Catalog

A source-grounded catalog of Apple Core AI models — provenance, licenses, benchmarks, and recommendations. Not affiliated with Apple.

Quick Start

pip install coreai-catalog
coreai-catalog recommend --task "private OCR on iPhone" --license likely

from coreai_catalog import Catalog

catalog = Catalog.load()
